Definig css

CSS is an acronym for Cascading Style Sheets.
It is a language to describe the format of HTML files and how to display their elements,
which means that CSS is an essential part of web design because it determines the way web pages are displayed and formatted,
and HTML can be written without CSS, but the way the elements are displayed will be simple, unformatted, and attractive, and CSS cannot be written without HTML CSS, in short, is a set of rules for formatting that lets the browser know how to display the elements in an HTML file, and helps the user interact with web pages.
