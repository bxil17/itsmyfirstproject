ربط ملف ال CSS بملف ال HTML

عندما تكون أكواد HTML و CSS في ملفات منفصلة ، يجب ربط الملفات. وإلا فلن يتمكن ملف HTML من تحديد موقع رمز CSS ولن يتم تطبيق التنسيق المحدد للصفحة.

و لحل هذه المشكلة يمكنك استخدام عنصر < link> لربط ملفات HTML و CSS معًا.

يجب وضع العنصر < link> داخل رأس ملف HTML. إنها علامة ذاتية الإغلاق وتتطلب السمات التالية:

href - مثل عنصر الارتساء ، يجب أن تكون قيمة هذه السمة هي العنوان أو المسار إلى ملف CSS.

rel - تصف هذه السمة العلاقة بين ملف HTML وملف CSS نظرًا لأنك تقوم بالارتباط بstyle sheet ، يجب تعيين القيمة على style sheet و تكون هذة القيمة ثتبتة دائما عند ربط ملفات ال css بال html.

عند ربط الملفان معًا ، سيبدو العنصر < link> كما يلي:
